What is a Privly Message?

Message is a privacy application on Privly. Privly Messages encrypt your content with a key that is attached to the link you share across the web. If someone has access to the server where the content is stored, and to the link you create, they will be able to decrypt your content. You can use Markdown to format your posts. You can read more about Privly Applications or about Messages specifically.
